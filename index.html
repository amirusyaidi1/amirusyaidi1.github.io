UniRun

ABOUT THE RACE


We are proud to anounce for the firt time UniRun. This is a fun run which is only a 5km easy run that can be participated by any age. This run course traverrse through the beautiful and scenic landscape of IIUM Gombak. The run will start and end at Stadium Saidina Hamzah. We choose only 5km because we want to show that everyone deserve and should exercise and sport are not for the talented one only.

the targeted participant are mostly from public univerities across Malaysia. This is because we know that university life is hard because student spent their time studying and doing assignments and hardly doing any exercise at all. We believe that to excell in studies, student also must be physically good and healthy.

We invite you to be part of this exciting fun run at IIUM and thank you for your participation.



RACE INFO

DATE: 25 December 2018
VENUE: Saidina Hamzah Stadium
Distance Start Time End Time
5KM 8.00pm(25/12/2018) 11.00pm(25/12/2018)

RACE PARTICIPATION

The race is open to all men and women of all nationalities.
The organisers would like to remind all participants that even 5km run can be a suffering, and recommends that you should do some light training.
